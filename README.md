# 技术

- 前：vue3、vue-router、pinia、ts、less、axios、element-plus、vite、mavon-editor
- 后：springboot、mybatis-plus、maven、lombok、kaptcha、pagehelper

# 角色

- 游客
  - 查看、搜索博客、查看评论
  - 查看用户发布、顶过、踩过的博客
  - 登录、注册
- 普通用户
  - 包含游客所拥有的功能
  - 退出登录 、注销账号
  - 自动保存博客、删除博客
  - 顶、踩、收藏博客，保存浏览记录
  - 发布、回复、删除、顶、踩评论
- 管理用户
  - 包含游客和普通用户所拥有的功能
  - 博客管理：查看、审核、启用、禁用博客
  - 评论管理：查看、启用、禁用评论
  - 用户管理：查看、封禁、解封用户
  - 字典管理
